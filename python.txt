operator in python

1.Arithmetic Operators
    + (Addition)   a=5+4
    - (Subtraction)
    * (Multiplication)
    / (Division - returns a float)
    // (Floor Division - returns an integer) 
    % (Modulus/Remainder)
    ** (Exponentiation)

2.Comparison (Relational) Operators
     == (Equal)
     != (Not equal)
     > (Greater than)
     < (Less than)
     >= (Greater than or equal)
     <= (Less than or equal)

3.Logical Operators 
     and (True if both are true)
     or (True if one is true)
     not (Reverses the boolean state)

4.Bitwise Operators
     & (AND)
     | (OR)
     ^ (XOR) 
     ~ (NOT)
     << (Left Shift) 
    >> (Right Shift)

5.Membership Operators: 
     in (True if found)
     not in (True if not found).

6.Identity Operators: 
     is (True if identical)
     is not (True if not identical)
